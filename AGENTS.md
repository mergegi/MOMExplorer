# AGENTS.md

## Rules must follow when developing the project
* Don't use complex frameworks, use lightweight libraries is fine.
* Use open source or free resource as much as possible.
* Don't use paid API, if you need to use API, try to find free API or use mock data.
* Do step by step. For each step, create a commit with short and clear message describing the change.
* Push the code to GitHub after finishing all the steps.
* Keep a design document in design.md under project root to record the design decisions and architecture of the project. Make sure update it whenever you make significant changes to the design or architecture.

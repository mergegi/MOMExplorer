# MOMExplorer Website Design

## Goal
Build an educational interactive website that simulates a Monmouth County, NJ map containing key historical locations, with mini-games to make learning fun.

## Architecture
- `index.html` : main page
- `styles.css` : layout and visual styling
- `script.js` : map interactions, location details, and game logic

## Map UI
- Map container with silhouette shape
- Clickable markers placed via percentage coordinates
- Tooltip on marker hover
- Info panel shows details when location selected

## Mini-games
- Location-specific trivia
- Modal-based quiz interface
- Score tracking and end-of-game result message

## Further extension ideas
- Add a second mini-game (matching, drag-and-drop timeline)
- Use external map tiles and accurate coordinates
- Add backend to persist user scores and progress

# MOMExplorer: Monmouth County Historical Map

A small static website project to explore Monmouth County, NJ history with interactive map markers and mini-games.

## Features
- simulated county map with clickable historical sites
- site details shown in info panel
- location-themed trivia mini-games
- responsive layout for desktop and mobile

## Run locally
1. Open `index.html` in your browser
2. Click a red circle marker
3. View historical details and click "Play Mini-Game"
